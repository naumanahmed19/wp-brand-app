== Description ==

WP Brand Blocks helps you to create your android and ios app using xapp.
This plugin requires xapp to work, intall it first.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wp-brand-blocks` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
